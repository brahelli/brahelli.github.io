Clone Prototype

This is one of the simpler projects that I have been working on. It consists of a player, controllable by a mouse and keyboard or gamepad, being able to move around and jump in a smooth fashion.

However, the focus of this project was to practice inheritance and how it could affect gameplay.​

As you can see, when the player holds down a particular button, time slows down and are able to select an entity.​

Once they release said button, they change into that entity, taking on the abilities and appearance of it.​

To accomplish this, I wrote one script that controlled how entities moved on the screen, and other behaviours such as health and damage, which you cannot see in this video, as this is an old version.​

Then I created more scripts for each entity that specifies exactly what they can do but inherits from the larger script to get the correct movement and behaviours.​

This then allows the player to simply change the class that the player entity is using, hot swapping at runtime without interrupting the flow of gameplay.​

This project was more of a practice for inheritance as revision for an exam that I had coming up in Computer Science.​
